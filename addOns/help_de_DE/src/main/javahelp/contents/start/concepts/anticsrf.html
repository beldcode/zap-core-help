Anti CSRF Tokens

Anti CSRF Token sind (pseudo-)zufällige Parameter, die genutzt werden, um sich gegen Cross Site Request Forgery (CSRF) Angriffe zu schützen.
Jedoch erschweren sie die Tätigkeit des Penetrationstest, insbesondere, weil sie nach jedem neuen Form-Aufruf neu generiert werden.

ZAP erfasst Anti-CSRF-Token rein anhand der Attributbezeichner - die Liste der Attributbezeichner, die als Anti-CSRF-Token interpretiert werden, kann über den Options Anti CSRF screen konfiguriert werden. Sobals ZAP diese Token erfasst, wird der Wert des Tokens und die tokenerzeugende URL gesichert.
Andere Scanner, wie active scanner weisen Optionen auf, bei denen ZAP die Token automatisch wiederherstellt, falls dies nötig ist.
