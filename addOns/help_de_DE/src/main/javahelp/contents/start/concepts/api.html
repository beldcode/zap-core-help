API

ZAP stellt ein Application Programming Interface (API) zur Verfügung, welches die programmatische Interaktion mit ZAP ermöglicht.

Die API unterstützt JSON, HTML und XML Dateiformate. Ein einfaches Web UI erlaubt die Erkundung und Nutzung der API. Es ist unter http://zap/ erreichbar, sobald ZAP oder der Host als Proxy an bestimmten Ports eingesetzt werden, die ZAP belauscht, wie zum Beispiel http://localhost:8080/.

Standarmäßig ist nur der Rechner, auf dem ZAP läuft, in der Lage, die API anzusprechen. Man kann jedoch auch anderen Maschinen, die in der Lage sind ZAP als Proxy zu nutzen, Zugang zur API verschaffen. Die API kann unter Options API screen konfiguriert werden

Die API erlaubt Zugriff auf die meisten ZAP Funktionalitäten, wie den active scanner and spider. Künftige Versionen von ZAP werden den Zugriff auf bestehende Funktionen weiter ausbauen.
