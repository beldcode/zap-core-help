Erweiterungen (Add-ons)

Add-ons erweitern ZAP um zusätzliche Funktionalitäten.
Add-Ons erhalten vollen Zugriff auf sämtliche Zap-Internals und können auf diese Weise wirksame neue Funktionen bereitstellen.
.
Sie können mit Hilfe des Manage Add-ons Dialoges.
Add-Ons dynamisch aus dem online Add-On Marketplace herunterladen. Add-Ons können in der Regel über das ZAP UI hinzugefügt und gelöscht werden, ohne dass die Anwendung neu gestartet werden muss.

Add-Ons stellen kontextspezifische Hilfe-Informationen zur Verfügung, welche dynamisch in die Hilfe-Seite integriert werden. Add-ons can provide context sensitive help which is dynamically integrated with the help file.
Falls Sie diese Seite über die ZAP help file aufgerufen haben (im offline Modus), sollten alle Hilfe-Seiten im Zusammenhang mit installierten Add-Ons unter here.
erreichbar sein.

Add-ons können folgende Zustände aufweisen:
     Releasebestätigt hohe Qualität und eine angemessene Brauchbarkeit für die Anwendungszwecke
     Betaweist eine gute Qualität nach, jedoch sind Verbesserungen und/oder Tests notwendig
     Alphadeutet auf einen noch frühen Entwicklungsstand hin

Speicherorte der Add-Ons, welche der ZAP-Anwendung zur Verfügung stehen

Um ZAP Add-Ons zugänglich zu machen, müssen diese an einem der folgenden Orte gespeichert werden:
  • plugin Ordner im ZAP installation Ordner;
  • plugin Ordner im ZAP user Ordner.
